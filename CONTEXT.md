# Giffgaff Guard

## 领域术语

### 保号 (Keeper)
通过产生一次移动数据流量，使 Giffgaff SIM 卡被识别为"已使用"，从而延长号码有效期。

### 档位 (Tier)
单次保号尝试消耗的预设流量大小。当前定义：
- **测试档 (Test)**: 10 KB — 探索最低有效阈值
- **标准档 (Standard)**: 60 KB — 推荐日常档位
- **安全档 (Safe)**: 120 KB — 与原版一致，确保成功

### 保号成功 (Keeper Success)
用户通过 `*100#` 查询余额，确认流量被计费。应用本身无法直接判定。

### 档位升级 (Tier Escalation)
当低档位保号未成功时，手动提升到更高档位再次尝试的过程。

## 边界

- 应用不连接任何第三方 API
- 所有数据存储在浏览器本地 (localStorage / IndexedDB)
- 必须在移动数据网络下使用，Wi-Fi 不会产生计费
